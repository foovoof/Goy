# Goy — منصة التحقيق والاستخبارات المفتوحة (OSINT)

منصة ثابتة (HTML/CSS/JS) لجمع المعلومات، التحري، والاستعلام من مصادر مفتوحة (OSINT)
وربطها ببعضها. تدعم البحث عن: أسماء المستخدمين، البريد، الهاتف، IP/النطاقات،
الصور، الوثائق، والشبكات الاجتماعية عبر عشرات الخدمات في نقرة واحدة.

## المزايا
- شريط بحث موحد يوزّع الاستعلام على أفضل أدوات الـ OSINT
- تصنيف ذكي حسب نوع المُدخل (تلقائي: بريد، هاتف، IP، دومين، اسم مستخدم)
- روابط جاهزة لأكثر من 60 خدمة/دورك بحثي
- مفضلة محلية (localStorage) + سجل استعلامات
- واجهة عربية RTL، تصميم داكن، بدون أي خادم أو قاعدة بيانات

## التشغيل محلياً
```bash
# أي خادم ملفات ثابتة
python3 -m http.server 8080
# أو
npx serve .
```
ثم افتح: http://localhost:8080

## النشر على Render (Static Site)
يوجد `render.yaml` في الجذر. الخطوات:
1. ادخل [dashboard.render.com](https://dashboard.render.com/) → **New** → **Blueprint**.
2. اربط مستودع GitHub `foovoof/Goy`.
3. سيقرأ Render ملف `render.yaml` وينشئ خدمة **Static Site** تلقائياً.
4. اضغط **Apply** — سيصبح الموقع متاحاً خلال دقيقة على رابط `*.onrender.com`.

بديل يدوي: **New → Static Site**، Build Command: (اتركه فارغاً)، Publish Directory: `.`.
