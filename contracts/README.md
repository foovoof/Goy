# عقود منصة Goy

هذا المجلد يحوي كل عقود البيانات (JSON Schema)، وواجهات REST (OpenAPI)، وقواميس الأحداث (YAML) الخاصة بمنصة تحقيق واستخبارات مفتوحة المصدر Goy.

## البنية

- `agent/` — عقود الوكلاء الذكيين (Agents) ومهامهم
- `api/` — واجهات REST داخلية وعامة
- `capabilities/` — تعريف القدرات والمعالجات والمزودين
- `common/` — عقود مشتركة (معرفات، طوابع زمنية، تصفح…)
- `confidence/` — نماذج مستويات الثقة والموثوقية
- `domain/` — الكيانات الجوهرية (تحقيق، دليل، هوية، علاقة…)
- `entities/` — الكيانات الاستخباراتية (شخص، منظمة، مكان…)
- `events/` — قواميس الأحداث بين الخدمات
- `evidence/` — الأدلة وسلاسل الحيازة
- `graph/` — نموذج الرسم البياني للعلاقات
- `identity/` — الهويات الرقمية والدمج والمطابقة
- `investigations/` — التحقيقات والفرضيات والنتائج
- `messages/` — رسائل CQRS (أوامر/استعلامات/أحداث)
- `orchestration/` — سير العمل وحالات التنفيذ
- `provenance/` — تتبع المصدر والحيازة
- `registry/` — سجل العقود والتبعيات
- `relationships/` — أنواع العلاقات
- `security/` — الأذونات والأدوار والأسرار
- `service-interfaces/` — واجهات الخدمات الداخلية
- `validation/` — قواعد التحقق والفحص
- `versioning/` — سياسات الإصدار والترقية
- `workflow/` — تعريفات سير العمل

كل ملف مخطط مطابق لـ JSON Schema Draft 2020-12، وكل ملف OpenAPI مطابق للإصدار 3.1.
